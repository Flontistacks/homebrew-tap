# Flontistacks Homebrew Tap

Install [adl](https://github.com/Flontistacks/adl) with one command:

```bash
brew install Flontistacks/tap/adl
```

Upgrade to the latest stable release:

```bash
brew update
brew upgrade Flontistacks/tap/adl
```

Latest development build:

```bash
brew install --HEAD Flontistacks/tap/adl
```

See the [adl README](https://github.com/Flontistacks/adl#readme) for usage, configuration, keybindings, and the optional macOS app launcher.
